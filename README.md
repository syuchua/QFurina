## 简介

QFurina 是一个基于 Python 的 QQ 机器人后端服务，提供了一系列自动化和交互功能。它支持多种消息平台，并且具有强大的插件系统，允许用户轻松扩展和定制功能。

## 主要特性

- AI 聊天和角色扮演
- 群聊管理
- 图片生成和发送
- AI 绘画
- 语音生成
- 天气查询
- 联网搜索
- 定时任务
